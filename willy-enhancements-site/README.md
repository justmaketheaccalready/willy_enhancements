# willy.enhancements

A minimal, dark React site built with Vite + Tailwind + Framer Motion.

## Local dev
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
- Push to GitHub and connect to Vercel/Netlify (Framework: **Vite**).
- Build command: `npm run build`
- Output dir: `dist`
